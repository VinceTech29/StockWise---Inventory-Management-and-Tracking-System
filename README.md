# StockWise - Inventory Management and Tracking System

## Description
StockWise is a simple Java-based inventory management and tracking system designed for small businesses. It allows users to manage products, track stock levels, and perform basic inventory operations efficiently. This project was created as the **Java 103 Final Project in 2024**.

## Features
- Add, update, and delete products
- Track inventory quantities
- Search products by name or ID
- Display all products in the inventory
- Simple console-based interface
